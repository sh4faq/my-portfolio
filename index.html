Whoami

A short profile, in the writer's own words.

Based in New York. English at work, Arabic at home. I've been writing code for about a decade, and most of it started because somebody else's software was in my way.

Burp Suite was a GUI. I wrote an API on top of it. Recon was getting repetitive, so I shipped twelve tools into MCP Kali Server and let an AI run them for me. The maintainer reviewed it and merged. Red Alert 2's voxel format had been undocumented forever. I documented it on a Saturday and shipped a Blender exporter on Sunday. None of this was assigned. I just don't like waiting.

Finishing CS at CUNY Lehman in May 2026. I read disassembly for fun and write CRUD apps for money. People keep asking me which side I want to be on. So far the answer is yes.
